# Arquivo de Tarefas

Tarefas concluidas (`done`) ou canceladas (`cancelled`) sao movidas para este diretorio.

## Regras

1. **Manter o ID original** — nunca renumerar uma tarefa arquivada
2. **Adicionar status final** no topo do arquivo: `status: done` ou `status: cancelled` com data
3. **Remover da index.md** ativa e adicionar na secao "Arquivadas recentes"
4. **Nao editar conteudo** — o arquivo e um registro historico
5. **Nome do arquivo:** manter o mesmo nome que tinha em `active/`
